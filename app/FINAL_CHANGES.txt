3. ✅ ĐỔI FORMAT THỜI GIAN: YYYY-MM-DD HH:mm

   Files đã cập nhật:
   • src/screens/History/HistoryScreen.js ✓
   • src/screens/Home/ChartScreen.js ✓
   
   Format cũ: DD/MM/YYYY HH:mm (01/01/2026 00:00)
   Format mới: YYYY-MM-DD HH:mm (2026-01-01 00:00)
   
   Parse function:
   const match = str.match(/^(\d{4})-(\d{2})-(\d{2})\s+(\d{2}):(\d{2})$/);
   const [, year, month, day, hour, minute] = match;
   return new Date(year, month - 1, day, hour, minute);
